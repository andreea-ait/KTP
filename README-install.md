How to install:

https://reactjs.org/docs/create-a-new-react-app.html

1. install `Node >= 14.0 npn >= 5.6` from https://nodejs.org/en/
2. To create a project, run:
  ```console
  npx create-react-app my-app
  cd my-app
  ```
3. To start a project, run 
  ```console
  npm start
  ```
